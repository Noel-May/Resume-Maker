# Resume-Maker
AI Resume Builder
Resume Maker is an interactive web application that allows users to create professional, ATS-friendly resumes in minutes. 
Users can fill in their personal information, skills, education, certificates, and languages, choose from three sleek templates, preview their resume in real-time, match it with job descriptions, and export as PDF or DOCX.
## Features
- Interactive resume form with live preview
- Three stylish resume templates (Black, Silver, Gold)
- Skills displayed as animated badges
- Save and load drafts using local storage
- Job description matching with percentage score
- Export resumes as PDF or DOCX
- Fully responsive and modern design
## How to Use
1. Clone or download this repository.
2. Open `index.html` in a browser **or** use VS Code Live Server for full functionality.
3. On the landing page, click the **Start Creating** button.
4. Fill in your details in the form.
5. Select a template to see your live preview.
6. Optionally, paste a job description to see how well your resume matches.
7. Save draft, load draft, or export as PDF/DOCX when ready.
Live Demo : https://noel-may.github.io/Resume-Maker/
## Technology Stack
- HTML5
- CSS3
- JavaScript (ES6)
- Libraries: html2canvas, jsPDF, Font Awesome
